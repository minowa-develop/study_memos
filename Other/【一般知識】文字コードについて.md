# 【一般知識】文字コードについて

## 文字エンコーディング(文字符号化方式): コードポイントから符号に変換する方法

- UTF-8,16,32
- ENC-JP
- shift_jis系(ms932,cp932)
- ISO-2022-jp

## 文字コード(符号化文字集合,文字セット,キャラクターセット):文字と文字に割り当てた番号(コードポイント？)の対応表

- JIS X 0201
- JIS X 0208
- JIS X 0213
- ISO/IEC 10646(UCS-2, UCS-4)
- unicode

## 流れ

文字 → コードポイント → 符号

## 実例

### utf8例

入力文字: あ
unicode: U+3042
utf8: E3 81 82

### cp932例

入力文字: あ
JIS X 0208: 04-02
cp932: 82A0
